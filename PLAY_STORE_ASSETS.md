## 3. Phone Screenshots (At least 2) ‚≠ê

**Requirements:**
- Minimum 2, maximum 8
- Aspect ratio: 16:9, 9:16, or similar
- Recommended: 1080x1920 pixels
- Format: PNG or JPEG
- Max file size: 8MB each

**What to Screenshot:**
1. **Dashboard** - Main screen showing overview
2. **Finance** - Transaction list or chart
3. **Goals** - Goals overview with progress
4. **Tasks** - Task management view
5. **Dark Mode** - Show theme toggle

**How to Capture:**

### Desktop Browser (Chrome):
1. Open app: http://localhost:3000
2. Press `F12` (DevTools)
3. Click device toolbar (phone icon)
4. Select: iPhone 12 Pro (390x844)
5. Take screenshot
6. Resize to 1080x1920

### Android Device:
1. Open app on phone
2. Take screenshots (Power + Volume Down)
3. Transfer to computer

**Enhancement (Optional):**
- Add device frames: https://mockuphone.com/
- Add text overlays with features
- Use screenshot editing tools
